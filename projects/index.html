Projects Page

I currently am working on two projects. Right now my side projects focus on learning some of the graphical capabilities of Python. I want to learn some fo the tools used for game dev, but I want to do so in python as I use it so much for quick fixes and work.

Chess Bot

I’m creating a chess bot for discord. This bot is using PIL to render a chess board. The game state will likely be in postgreSQL since I’d like to try installing it. Github to come.

Underfoot

I’m working on a 2d game. The assets are what I’m currently learning how to do and will be drawing them in asprite. The actual game is going to run on pygame. I’m doing the classes in jupyter notebooks while I learn about the structures you need for a game and then I will be using NBDev to convert them to standard python modules.
